## 01. selection sort (선택 정렬)
* 특징
  * 가장 작은 값을 선택해 제일 앞으로 보내는 알고리즘
  * 가장 기초적인 정렬 방법이다.
  * 시간 복잡도 = **O(N^2)**
    > ***cf. 원래는 N*(N+1)/2 수식이긴 하나 N값이 크다고 가정할땐 컴퓨터에선 '/2'와 '+1'와같은 값은 생략하기 때문**
